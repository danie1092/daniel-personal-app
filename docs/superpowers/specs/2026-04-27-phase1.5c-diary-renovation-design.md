## 결정 사항 (자동)

### 디자인
- **헤더**: 큰 날짜 (월/일/요일) — 기존 디자인 유지하되 토큰 적용
- **잔디 카드** 신규: 30일 dots, 감정별 색상 (filled = 작성, empty = 미작성)
- **오늘 일기 카드**: 감정 chip picker + textarea + 저장 버튼
- **과거 일기 리스트**: 카드 클릭 → 시트 슬라이드업 → 내용 보기/수정/삭제

### Server Components 마이그레이션
```
src/app/(main)/diary/
  page.tsx                       # async server, 데이터 fetch + 컴포넌트 조합
  DiaryGrass.tsx                 # server, 30-day dots
  DiaryInputCard.tsx             # client, emotion + textarea + save (sound + Server Action)
  DiaryPastList.tsx              # client, list items + click→sheet
  DiaryEditSheet.tsx             # client, view/edit/delete sheet
  emotionTokens.ts               # 8 감정 → emoji/label/bg 매핑
  actions.ts                     # upsertTodayDiary, updateDiary, deleteDiary
src/lib/diary/
  data.ts                        # getTodayDiary, getRecentDiaries, getMonthGrass
  data.test.ts                   # 4 tests
```

### Server Actions
```ts
upsertTodayDiary({ content, emotion }): { ok, id } | error
updateDiary(id, { content, emotion }): { ok } | error
deleteDiary(id): { ok } | error
```
